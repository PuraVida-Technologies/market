
# Babyation

```bash 
$ git clone git@github.com:Babyation/server.git
```
## Installation

```bash
$ npm install
```

## Start docker container for mongodb

```bash
$ docker-compose up
```

## Running the app

```bash
# Production
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Open the graphql playground

- [http://localhost:5000/graphql](http://localhost:5000/graphql)

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
